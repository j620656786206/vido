All user-facing documentation must be provided in both English and Traditional Chinese (zh-TW).

**Why:** Vido's core audience is Traditional Chinese NAS users, but Unraid's community is global. Both versions are needed for accessibility.

**How to apply:** When creating docs (installation guides, API docs, SSE docs), always produce two versions. File naming convention: `doc-name.md` (EN) + `doc-name.zh-TW.md` (zh-TW).
